# weather-forecast

Development of a website that displays the "Weather Forecast", using HTML, CSS and JS.

In HTML, you use basic tags to develop the structure of the site.

In CSS, you use the class concepts to change the appearance of the HTML elements.

In JS, uses the api of the site "open weather" to dynamically change the data of the "city name", "temperature", "climate" and "humidity".

In order to practice the knowledge about HTML, CSS and JS, the site was developed in Portuguese (Brazil).
